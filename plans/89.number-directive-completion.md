## Outline

### Goal
Add auto-completion for 10 number directives with simple numeric parameter placeholders.

### Definition: Number Directive

Number directives accept a single numeric parameter (integer or float). Some have constraints like minimum values or optional parameters. Example: `%%scale 0.8` or `%%barsperstaff 4`.

### Scope

Number directives (10 layout/formatting directives):
- `lineThickness` - Line thickness multiplier
- `stretchlast` - Stretch factor for last line (0-1, optional)
- `fontboxpadding` - Padding around font boxes
- `voicescale` - Voice scaling factor
- `scale` - Global scaling factor
- `barsperstaff` - Number of bars per staff line (min: 1)
- `measurenb` - Measure numbering interval (min: 0)
- `barnumbers` - Bar number display interval (min: 0)
- `setbarnb` - Set starting bar number (min: 1)
- `newpage` - Page number for new page (optional)

All 10 are defined in `parse/types/directive-specs.ts` with `params: [{ type: "number", ... }]`:
- Line 151: `lineThickness: { params: [{ type: "number" }] }`
- Line 152: `stretchlast: { params: [{ type: "number", optional: true, min: 0, max: 1 }] }`
- Line 185: `fontboxpadding: { params: [{ type: "number" }] }`
- Line 188: `voicescale: { params: [{ type: "number" }] }`
- Line 193: `scale: { params: [{ type: "number" }] }`
- Line 201: `barsperstaff: { params: [{ type: "number", min: 1 }] }`
- Line 206: `measurenb: { params: [{ type: "number", min: 0 }] }`
- Line 207: `barnumbers: { params: [{ type: "number", min: 0 }] }`
- Line 208: `setbarnb: { params: [{ type: "number", min: 1 }] }`
- Line 228: `newpage: { params: [{ type: "number", optional: true }] }`

Exclusions:
- Measurement directives are excluded because they use `type: "measurement"` instead of `type: "number"`
- MIDI directives (bassvol, chordvol, channel, beatmod, etc. at lines 301-317 in MIDI_COMMAND_SPECS) are excluded because they will be handled separately in plan 93 (MIDI directive completions) with their own context detection (`%%MIDI command` syntax)
- Font directives have a number parameter for font size, but they are handled separately by `getFontDirectiveCompletions()` which provides a complete multi-parameter snippet

### Snippet Format

```
%%scale ${1:1.0}
```

LSP Snippet Syntax:
- `${1:1.0}` creates an editable placeholder with default value `1.0`
- Unlike choice placeholders `${1|a,b|}`, this allows free-form numeric input
- The default value serves as a hint for typical usage

Default values per directive:
- `lineThickness` - `${1:1}` (multiplier, 1 = normal)
- `stretchlast` - `${1:0.8}` (typical stretch factor, 80% stretch is common)
- `fontboxpadding` - `${1:0.1}` (small padding)
- `voicescale` - `${1:1}` (no scaling)
- `scale` - `${1:1}` (no scaling)
- `barsperstaff` - `${1:4}` (common for hymns and simple tunes)
- `measurenb` - `${1:1}` (number every measure for maximum clarity)
- `barnumbers` - `${1:1}` (every bar)
- `setbarnb` - `${1:1}` (start at 1)
- `newpage` - `${1:}` (empty placeholder because parameter is optional)

Note on newpage: Because newpage's parameter is optional (can be omitted entirely), we use an empty default placeholder. Users can type a page number or press tab to skip, resulting in `%%newpage` with no parameter, which starts a new page using automatic numbering.

### Design Decisions

1. Use `InsertTextFormat.Snippet` with `${1:default}` editable placeholder
2. Default values chosen based on typical usage patterns
3. Respects `supportsSnippets` capability with plain text fallback
4. Use `CompletionItemKind.Keyword` to match other directive completions
5. Include descriptive documentation with valid range hints where applicable
6. Completion items returned after existing directive types

### 1.2 Add Snippet Generator and Completion Function

File: `abc-lsp-server/src/completions.ts`

Add after the number directive constants:

```typescript
export function buildNumberSnippet(defaultValue: string): string {
  return `\${1:${defaultValue}}`;
}

export function getNumberDirectiveCompletions(prefix: string, supportsSnippets: boolean = true): CompletionItem[] {
  return NUMBER_DIRECTIVES
    .filter((d) => d.label.toLowerCase().startsWith(prefix.toLowerCase()))
    .map((directive) => ({
      label: directive.label,
      kind: CompletionItemKind.Keyword,
      documentation: `${directive.documentation}. Format: %%${directive.label} <number>`,
      insertText: supportsSnippets ? directive.label + " " + buildNumberSnippet(directive.defaultValue) : directive.label,
      insertTextFormat: supportsSnippets ? InsertTextFormat.Snippet : InsertTextFormat.PlainText,
      data: { type: "number-directive", directive: directive.label },
    }));
}
```

// =============================================================================
// Number Directive Completions

describe("getNumberDirectiveCompletions", () => {
  it("returns all 10 number directives when prefix is empty", () => {
    const result = getNumberDirectiveCompletions("");
    expect(result).to.have.length(10);
    const labels = result.map((r) => r.label);
    expect(labels).to.include.members(NUMBER_DIRECTIVES.map((d) => d.label));
  });

  it("filters by prefix", () => {
    const result = getNumberDirectiveCompletions("scale");
    expect(result).to.have.length(1);
    expect(result[0].label).to.equal("scale");
  });

  it("filters case-insensitively", () => {
    const result = getNumberDirectiveCompletions("BARS");
    expect(result).to.have.length(1);
    expect(result[0].label).to.equal("barsperstaff");
  });

  it("uses snippet insert format when supportsSnippets is true", () => {
    const result = getNumberDirectiveCompletions("scale", true);
    expect(result[0].insertTextFormat).to.equal(InsertTextFormat.Snippet);
  });

  it("uses plain text insert format when supportsSnippets is false", () => {
    const result = getNumberDirectiveCompletions("scale", false);
    expect(result[0].insertTextFormat).to.equal(InsertTextFormat.PlainText);
    expect(result[0].insertText).to.equal("scale");
  });

  it("includes default value in snippet placeholder", () => {
    const result = getNumberDirectiveCompletions("scale", true);
    expect(result[0].insertText).to.include("${1:1}");
  });

  it("returns empty array when no directives match prefix", () => {
    const result = getNumberDirectiveCompletions("xyz");
    expect(result).to.have.length(0);
  });

  it("uses Keyword completion kind", () => {
    const result = getNumberDirectiveCompletions("");
    expect(result.every((r) => r.kind === CompletionItemKind.Keyword)).to.be.true;
  });

  it("includes data field with type and directive name", () => {
    const result = getNumberDirectiveCompletions("scale", true);
    expect(result[0].data).to.deep.equal({ type: "number-directive", directive: "scale" });
  });

  it("inserts space between directive name and snippet", () => {
    const result = getNumberDirectiveCompletions("scale", true);
    expect(result[0].insertText).to.match(/^scale \$\{1:/);
  });

  it("includes documentation for each directive", () => {
    const result = getNumberDirectiveCompletions("");
    expect(result.every((r) => typeof r.documentation === "string" && r.documentation.length > 0)).to.be.true;
  });

  it("documentation includes format hint", () => {
    const result = getNumberDirectiveCompletions("scale", true);
    expect(result[0].documentation).to.include("Format: %%scale <number>");

describe("number directive completion integration", () => {
  it("returns number directive completions when typing after %%", () => {
    const context = getDirectiveCompletionContext("%%scale", 7);
    expect(context.type).to.equal("directive-name");
    if (context.type === "directive-name") {
      const completions = getNumberDirectiveCompletions(context.prefix, true);
      expect(completions.some((c) => c.label === "scale")).to.be.true;
    }
  });

  it("returns multiple matching directives for common prefixes", () => {
    const context = getDirectiveCompletionContext("%%bar", 5);
    expect(context.type).to.equal("directive-name");
    if (context.type === "directive-name") {
      const completions = getNumberDirectiveCompletions(context.prefix, true);
      expect(completions.some((c) => c.label === "barsperstaff")).to.be.true;
      expect(completions.some((c) => c.label === "barnumbers")).to.be.true;
    }
  });

  it("returns number directives alongside other directive types for shared prefix", () => {
    const context = getDirectiveCompletionContext("%%voice", 7);
    expect(context.type).to.equal("directive-name");
    if (context.type === "directive-name") {
      const allCompletions = [
        ...getFontDirectiveCompletions(context.prefix, true),
        ...getMeasurementDirectiveCompletions(context.prefix, true),
        ...getNumberDirectiveCompletions(context.prefix, true),

      // voicefont (font), voicescale (number) match "voice"
      expect(labels).to.include("voicefont");
      expect(labels).to.include("voicescale");
    }
